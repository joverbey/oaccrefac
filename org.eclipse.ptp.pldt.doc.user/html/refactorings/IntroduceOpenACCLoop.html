Introduce OpenACC Loop

Description:

Parallelizes a loop by adding an OpenACC parallel or kernels loop directive to it.

Example:

  1. Select the desired for loop to be changed and choose Parallelize (OpenACC) under the Refactoring menu.
  2. Select either Parallel or Kernels to indicate the type of parallelization desired.

Restrictions:

  • There must not be any dependences carried by the loop to be parallelized
  • The loop may not have a pragma or kernel already attached to it.
  • Kernels will throw a warnings instead of error.
