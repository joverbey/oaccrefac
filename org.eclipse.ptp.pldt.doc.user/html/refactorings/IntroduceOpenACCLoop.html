Introduce Parallel

Description:

Parallelizes a loop by adding a parallel loop directive to it.

Example:

  1. Select the desired for loop to be changed and choose Parallelize (OpenACC) under the Refactoring menu.
  2. The resulting for loop.

    3. Use:

Restrictions:

  • There must not be any dependences carried by the loop to be parallelized
  • The loop may not have a pragma already attached to it.

IntroduceKernels

Description:

Parallelizes a loop by adding a kernel loop construct over it.

Example:

  1. Select the desired for loop to be changed and choose Kernel Loops (OpenACC) under the Refactoring menu.
  2. The resulting loop.

Use:

Restrictions: Has the same dependencies as Introduce Parallel Loops except it will throw a Warning instead of an error.

  • 1.The loop may not have a pragma or kernel already attached to it.
