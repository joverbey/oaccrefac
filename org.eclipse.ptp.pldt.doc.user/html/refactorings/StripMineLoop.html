Strip Mine Loop

Use:

Strip mining splits a single loop into a double loop nest. Often, after this is performed, one of the results loops will be parallelized.

Description:

Turns a single loop into two loops such that the array is iterated over in the same order, but in multiple small strips.

Example:

  1. Select the desired for loop to be changed and choose Strip Mine Loop from the Refactoring menu.
  2. Next you will be prompted for the size of the strip.
  3. The resulting for loop.

Restrictions:

None
