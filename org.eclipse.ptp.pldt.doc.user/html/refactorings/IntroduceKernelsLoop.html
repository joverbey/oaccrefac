Description:

Parallelizes a loop by adding a kernel loop construct over it.

Example:

  1. Select the desired for loop to be changed and choose Kernel Loops (OpenACC) under the Refactoring menu.
  2. The resulting loop.

Use:

Restrictions: Has the same dependencies as Introduce Parallel Loops except it will throw a Warning instead of an error.

  • 1.The loop may not have a pragma or kernel already attached to it.
