Description: Parallelizes a loop by adding a parallel loop construct over it.

Example:

  • 1. Select the desired for loop to be changed and choose Paralellize (OpenACC) under the Refactoring menu.
  • 2. The resulting for loop.

Use:

  • 1.There must not be any dependences carried by the loop to be parallelized
  • 2.The loop may not have a pragma already attached to it.

Restrictions:
