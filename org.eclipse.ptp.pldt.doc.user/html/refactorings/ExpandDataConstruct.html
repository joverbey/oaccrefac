Description: Expands a data construct to envelop as many statements as possible only stopping once it hits another openACC Region or something leaves the scope.

Example:

  1. Select the desired data construct to be expanded and choose Expand Data Construct under the Refactoring menu.
  2. The resulting construct.

Use: Expanding a data construct allows for more data to be copied in and copied out at one time.

Restrictions:

  • None
