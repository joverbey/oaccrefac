Expand Data Construct

Use: Expanding a data construct may allow more data to be copied in and copied out at one time.

Description: Expands a data construct to envelop as many statements as possible only stopping once it encounters another OpenACC Region or a variable's scope would need to change.

Example:

  1. Select the desired data construct to be expanded and choose Expand Data Construct under the Refactoring menu.
  2. The resulting construct.

Restrictions:

  • None
