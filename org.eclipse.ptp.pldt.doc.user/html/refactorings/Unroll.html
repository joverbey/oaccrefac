Description: Duplicates the body of a loop inside the loop and reduces the number of iterations by the amount of duplications.

Example:

  • 1. Select the desired for loop to be changed and choose Unrolling under the Refactoring menu.
  • 2. Next you will be prompted .
  • 3. The resulting for loop.

Use: Loop Unrolling can decrease performance time regardless of parallelization. The execution of a for loop is an expensive process that is almost always far more expensive than the loop bodies cost. Because of this duplicating the for loop body and decreasing number of for loop iterations will have an overall decrease in cost.

  • 1. Must have constant upper and lower bounds.
  • 2. Index variable may not be altered in the loop body.
  • 3. Loop body cannot be empty.
  • 4. Loop must be a counted loop
