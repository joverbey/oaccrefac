Merge Data Construct

Use: Reduces the number of copy ins and outs down by combining data constructs, therefore increasing effciency.

Description: Combines two corresponding data constructs together.

Example:

  1. Select the desired data construct to be merged and choose Merge Data Construct under the Refactoring menu.
  2. The resulting construct.

Restrictions:

  • The second data construct must be immediately following the first.
  • Both data constructs must be a compound statements.
